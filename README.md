KiriStore
Интернет-магазин на React 19 с TypeScript и Tailwind CSS.

Функции:
Каталог товаров с фильтрацией по категориям, поиском и сортировкой
Страница товара с описанием, характеристиками и отзывами
Корзина покупок с расчетом стоимости
Оформление заказа с применением промокодов
Список желаемого
Сравнение товаров (до 4 единиц)
История заказов
Личный кабинет
Форма обратной связи
Переключение светлой и темной темы

Технологии:
React 19
TypeScript
Vite
React Router DOM
Tailwind CSS
i18next
LocalStorage для хранения данных

Инструкция по запуску:
Установка Node.js
Команда npm install
Команда npm run dev
Открытие в браузере

Страницы:
Главная
Каталог
Товар
Корзина
Оформление заказа
Список желаемого
Сравнение
Заказы
Профиль
О магазине
Обратная связь

API:
Используется dummyjson.com для демонстрации товаров
Используется randomuser.me для картинок профилей пользователей в отзывах
